批量导入设备问题数据

CSV格式说明

请按照以下格式准备CSV文件：

  • 必需字段: 标题/问题 或 描述/原因/分析 (至少包含其中一个)
  • 可选字段: 设备类型, 阶段, 发现者, 发现时间
  • 支持中英文列名
英文列名 中文列名 说明 是否必需
title / problem / issue 标题 / 问题 问题标题 必需
description / reason / analysis 描述 / 原因 / 分析 问题描述 必需
equipment_type / device_type 设备类型 设备类型 可选
phase / stage 阶段 发现阶段 (design/development/usage/maintenance) 可选
discovered_by 发现者 发现问题的人 可选
discovered_at 发现时间 发现问题的时间 (YYYY-MM-DD) 可选
